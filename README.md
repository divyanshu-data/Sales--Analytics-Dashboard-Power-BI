# Sales Analytics Dashboard (Power BI)

Interactive Power BI dashboard analyzing 1000+ retail 
sales transactions across 5 product categories, built 
to uncover insights on customer behavior, sales trends, 
and demographics.

## Key Insights
- Total Sales: 574K | Total Cost: 269K | Transactions: 1126
- Avg Order Value: ₹509.77 | Avg Basket Size: 2.50
- Repeat customers: 185 vs One-time: 84
- Top performing categories: Groceries & Electronics
- Age group 31-40 contributes highest sales

## Dashboard Features
- Sales & avg basket size by product category
- Sales breakdown by gender
- Sales by age bucket
- Customer type (repeat vs one-time)
- Monthly sales trend analysis
- Category-wise sales matrix

## Tools & Skills Used
Power BI, DAX, Data Modeling, Data Visualization, 
Data Cleaning

## Dataset
Retail sales transaction dataset (CSV) included in 
this repo.
